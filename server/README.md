# Watchlist Server

This is the server side of Watchlist project. This project was made using Spring Boot Framework and Java 8.

## Building project
- On root directory:
```
$ mvn package
```

## Running Server
- Go to `target` folder:
```
$ cd target
```

- Run Built jar file:
```
$ java -jar watchlist-0.0.1-SNAPSHOT.jar
```
